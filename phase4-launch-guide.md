# Phase 4: Launch & Closing (Week 4)

## Week 4 Goals
- Convert demo leads into paid users
- Close first 5-10 customers
- Establish initial revenue stream

## Conversion Tactics

### 1. Post-Demo Follow-up (Within 24 hours)
Email template:
```
Subject: Quick follow-up + 20% launch discount

Hi [Name],

Thanks again for your time today. As discussed, here's what we covered:
- Time savings with Drug Search
- Interaction checking workflow
- Rx-upload document analysis

To thank you for being an early adopter, I'm offering 20% off the first 3 months.
Use code: LAUNCH20

Ready to get started? Visit /pricing or reply to set up your free trial.

Best,
[Your name]
```

### 2. Team Trial Setup
- Get decision maker email
- Create team account with 5 free licenses
- Schedule 15-minute setup call

### 3. Pricing Options
- Free Plan: Basic tools, 5 searches/day
- Professional: $29/month (normally $39)
- Enterprise: Custom pricing with API access

### 4. Closing Sequence
```
Day 1: Demo call completed
Day 2: Thank you email + discount
Day 4: Gentle reminder if no response
Day 7: Final check-in
Day 14: If interested but no decision - schedule with procurement
```

## Success Metrics for Week 4
- Target: 5-10 paying customers
- Minimum viable: 3 customers
- Revenue goal: $1,000-$2,000 (first month)

## Ready Assets
- /pricing/ - Live pricing page
- /demo/ - Interactive demo
- /tools/rx-upload/ - Medical Document Analyzer
- All API endpoints working (Drug Search, Interactions, etc.)
